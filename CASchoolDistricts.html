Information about the project


Purpose


The purpose of this project is to create an interactive map with all K-12 public school districts on CA map with certain information showing as the user click on it. I have used district name, enrollment,dropout count, English Learner Counts, Drop Out percentages.... The geojason file I have pulled from ca.gov site has much more information about the districts and I can add any of that infromation upon request.
