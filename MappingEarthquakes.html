Project Example 2

Here is a page created during the Berkeley Data Analysis Bootcamp as one of the challanges. To create this page I have used the following tools ;
  • HTML,
  • CSS,
  • JavaScript,
  • Visual Studio Code,
  • Github,
  • Mapbox
  • Leaflet

Information about the project


Purpose

The purpose of this project is to visually show the differences between the magnitudes of earthquakes all over the world for the last seven days also trying to show their relations to techtonic plates.

Click here to see the json data

To complete this project, used a URL for GeoJSON earthquake data from the USGS website and retrieve geographical coordinates and the magnitudes of earthquakes for the last seven days. Then add the data to a map.

The Approach

Our approach was to use the JavaScript and the D3.js library to retrieve the coordinates and magnitudes of the earthquakes from the GeoJSON data. Then using the Leaflet library to plot the data on a Mapbox map through an API request and create interactivity for the earthquake data. This map has different layers of data and you can see it via street view or satellite view.

  • 1 layer has tectonic plate data
  • 2 Major Earthquake
  • 3 Earthquakes
