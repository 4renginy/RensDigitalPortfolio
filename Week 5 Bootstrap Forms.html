Forms using Bootstrap

The Bootstrap framework simplifies the styling of form elements. Bootstrap includes support for all standard form controls as well as new HTML5 input types such as datetime, number, email, url, search, tel, and color to name some.
Bootstrap also provides three different types of overall form layouts:
  • Vertical Form (default form layout)
  • Horizontal Form
  • Inline Form

Bootstrap Buttons + Decoration

Buttons are used for purposes like submitting forms, or interactions like showing or hiding something on a page. The Bootstrap button CSS provides a quick and easy way to create and customize buttons.

Making Buttons with Bootstrap

Button styles can be applied to any element. However, these classes are usually applied normally to anchor, button and input elements (a>, button> and input>).

The table below identifies the button classes that are available in Bootstrap:
Button Style Classes to Apply Description
Primary btn btn-primary Indicates primary action button in a group.
Info btn btn-info Often used as alternative to default button.
Success btn btn-success Indicates a successful or positive action.
Warning btn btn-warning Caution should be taken.
Danger btn btn-danger Dangerous or negative action.
Link btn btn-link Looks like a text link.
